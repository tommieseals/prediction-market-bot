📝 What It Does

Deploy any GitHub repo with one URL. Paste a link, Sidekick detects the project type, generates a Dockerfile, builds the container, configures reverse proxy, and issues SSL certificate. Self-hosted Vercel/Heroku alternative.

🧠 Key Features

  • Auto-detection of project types
  • Dynamic Dockerfile generation
  • Automatic SSL via Lets Encrypt
  • Nginx reverse proxy config
  • Container orchestration

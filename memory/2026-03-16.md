## The Exact Working Code

### Step 1: Find the right sidepanel target
```python
import json
import asyncio
import websockets

async def find_approve(target_id):
    ws_url = f'ws://localhost:9223/devtools/page/{target_id}'
    async with websockets.connect(ws_url) as ws:
        cmd = {
            'id': 1,
            'method': 'Runtime.evaluate',
            'params': {
                'expression': '''
                (function() {
                    const all = document.querySelectorAll('button');
                    for (const btn of all) {
                        const text = (btn.textContent || '').toLowerCase();
                        if (text.includes('approve')) {
                            return {found: true, tag: btn.tagName, text: btn.textContent.trim()};
                        }
                    }
                    return {found: false};
                })()
                ''',
                'returnByValue': True
            }
        }
        await ws.send(json.dumps(cmd))
        response = await ws.recv()
        result = json.loads(response)
        return result.get('result', {}).get('result', {}).get('value', {})
```

## Why pyautogui Failed

1. **Claude extension runs in separate Chrome renderer process**
2. **OS-level mouse events hit Chrome window BUT...**
3. **JavaScript event handlers in the extension don't fire**
4. **The sidepanel is in its own DOM context**

## Why CDP Worked

1. **CDP injects directly into V8 runtime**
2. **Runtime.evaluate executes JS in the page context**
3. **button.click() triggers the actual event handlers**
4. **Works regardless of window focus or position**

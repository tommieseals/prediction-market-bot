## 🔥 MiroFish Simulation Runner Fix (MAJOR)

### Problem Identified
- Simulations were hanging indefinitely (12+ hours stuck at round 9/24)
- Root cause: `await env.step(actions)` calls had **NO TIMEOUT**
- When Ollama connection dropped (`httpx.ReadError`), the async call hung forever
- 15+ simulations stuck in "running" status, never completing

### Root Cause Analysis
```
Simulation starts → Round 9 → Ollama connection drops → ReadError
→ env.step() hangs forever → Process becomes zombie → Never updates state
```

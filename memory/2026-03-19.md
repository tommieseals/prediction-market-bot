# 2026-03-19 Session Notes

## Mac Mini Gateway Restart Issue - DIAGNOSED ✅

**Problem:** Clawdbot gateway on Mac Mini restarts every 15 minutes exactly.

**Root Cause:** `com.clawd.auto-deploy` launchd job runs `check-deploy.sh` every 900s. The script compares local vs remote git hashes - but Mac Mini has **7 unpushed commits**, so `LOCAL != REMOTE` triggers deploy every time.

**Evidence:**
- `Your branch is ahead of 'origin/main' by 7 commits`
- 10+ auto-stash entries from repeated deploys
- Deploy log shows "Already up to date" after pull (local ahead, not behind)

**Fix Options:**
1. `git push origin main` (push the commits)
2. `launchctl unload ~/Library/LaunchAgents/com.clawd.auto-deploy.plist` (disable)
3. Fix check-deploy.sh logic to only deploy when remote is AHEAD
